# Delightful-Universe
Extends the track that plays at the end of cell stage (Brian eno - Delightful universe) to it's full, seven minute long version. Does not effect the cutscene sting version of the same track. 
Synergizes very nicely with share safe music, but not compatible with STREAM safe music as it effects the same song. Don't use if streaming this will give you an even bigger copyright strike than the trimmed version in vanilla
